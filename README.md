Project Description

MineOps is a mobile application designed to improve safety and communication in mining operations. The system replaces paper-based hazard logs with a real-time digital platform that allows workers to report hazards and enables supervisors to monitor safety conditions efficiently. The application uses Firebase Realtime Database to ensure instant synchronization of hazard data across all users.
